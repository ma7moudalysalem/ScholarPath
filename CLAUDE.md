# Claude Code conventions for ScholarPath

## Git commits

**Do NOT add a `Co-Authored-By:` trailer for Claude / Anthropic to any commit.**

The repo's commits should be attributed to the human author (Git config) only.
Skip the `Co-Authored-By: Claude ...` line that the default Claude Code commit
template suggests.

This is a graduation project — work attribution stays with the team members
listed in the README.
